VANQUISHER

Role: Gameplay Programmer

I worked as a Gameplay Programmer using C# in Unity. I developed the third-person player controller including movement mechanics such as running, jumping, and directional control.

I also designed and implemented the combat system, handling attack logic, input handling and combat interactions while integrating player animations for smooth gameplay.

Key Contributions

  • Developed third-person player movement system
  • Implemented combat mechanics and attack system
  • Integrated character animations
  • Built player controller using C# in Unity

Game Overview

A high-stakes third-person RPG set in the ancient Mayan kingdom where a young warrior must face brutal trials and become the ultimate Vanquisher.
